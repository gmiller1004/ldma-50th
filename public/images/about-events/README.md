# About Events page images

Optional photos for the **About LDMA Events** page (`/about-events`). If these files are missing, the page uses fallback images from the Stanton campground gallery.

## Suggested files

| File | Suggested content |
|------|-------------------|
| `event-1.jpg` | LDMA event — mining activity (e.g. sluicing, panning, or group at a claim) |
| `event-2.jpg` | Detector day or metal-detecting demo |
| `event-3.jpg` | Camp gathering (Dirt Fest, Gold N BBQ, or similar) |

Use landscape orientation and good contrast for the dark theme. The page displays them in a 3-column gallery; recommended size ~800×600px or larger.
